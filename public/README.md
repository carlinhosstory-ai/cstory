Pasta `public` — arquivos estáticos servidos publicamente.

Conteúdo:

- `index.html` -> redireciona para o `index.html` na raiz do projeto.
- `robots.txt` -> regras para robôs de busca.
- `images/` -> lugar para imagens públicas (atualmente com `.gitkeep`).

Uso local: sirva a pasta raiz com um servidor HTTP (ex.: `python -m http.server`).
